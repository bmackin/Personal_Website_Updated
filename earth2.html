EARTH2

The concept behind Earth2 was to create a platform for news websites that generally do not get as much exposure as websites that exist within the “Western media” circle. By using the Google Earth API (as well as Bing’s News API to extract news articles based on latitude and longitude), we were able to create an interactive newspaper where the stories are not curated, but are purely defined based on what location the user selects.
